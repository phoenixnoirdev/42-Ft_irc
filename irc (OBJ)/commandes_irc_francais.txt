1. 🔐 COMMANDES DE CONNEXION
===============================================================================

NICK <pseudonyme>
─────────────────
📝 Description : Définit ou change votre pseudonyme sur le serveur
🎯 Utilisation : NICK MonPseudo
📋 Exemples :
   NICK Alice
   NICK Bob123
⚠️ Remarques : 
   - Le pseudonyme doit être unique sur le serveur
   - Certains caractères spéciaux peuvent être interdits
   - Maximum généralement 30 caractères

USER <nom_utilisateur> <mode> <unused> <nom_réel>
─────────────────────────────────────────────────
📝 Description : Fournit les informations utilisateur au serveur
🎯 Utilisation : USER alice 0 * :Alice Dupont
📋 Exemples :
   USER alice 0 * :Alice Dupont
   USER bob 8 * :Robert Martin
⚠️ Remarques : 
   - Requis pour l'authentification complète
   - Le mode est généralement 0 ou 8
   - Le nom réel peut contenir des espaces

PASS <mot_de_passe>
───────────────────
📝 Description : Fournit le mot de passe du serveur (si requis)
🎯 Utilisation : PASS motdepasse123
📋 Exemples :
   PASS serveur_secret
⚠️ Remarques : 
   - Doit être envoyé avant NICK et USER
   - Nécessaire uniquement si le serveur est protégé par mot de passe

QUIT [message]
──────────────
📝 Description : Déconnecte proprement du serveur
🎯 Utilisation : QUIT :Au revoir !
📋 Exemples :
   QUIT
   QUIT :À bientôt !
   QUIT :Maintenance serveur

===============================================================================
2. 🏠 COMMANDES DE CANAL
===============================================================================

JOIN <#canal> [clé]
───────────────────
📝 Description : Rejoint un canal IRC
🎯 Utilisation : JOIN #general
📋 Exemples :
   JOIN #general
   JOIN #privé motdepasse
   JOIN #aide
⚠️ Remarques : 
   - Les noms de canaux commencent par # ou &
   - Une clé peut être requise pour les canaux protégés
   - Vous serez automatiquement opérateur si vous créez le canal

PART <#canal> [message]
───────────────────────
📝 Description : Quitte un canal
🎯 Utilisation : PART #general :À bientôt !
📋 Exemples :
   PART #general
   PART #aide :Problème résolu, merci !

LIST [#canal]
─────────────
📝 Description : Liste les canaux disponibles sur le serveur
🎯 Utilisation : LIST
📋 Exemples :
   LIST
   LIST #general
⚠️ Remarques : Affiche les canaux publics et leur nombre d'utilisateurs

NAMES [#canal]
──────────────
📝 Description : Liste les utilisateurs dans un canal
🎯 Utilisation : NAMES #general
📋 Exemples :
   NAMES #general
   NAMES
⚠️ Remarques : 
   - Sans paramètre, liste tous les utilisateurs visibles
   - Montre les privilèges : @ pour opérateur, + pour voice

===============================================================================
3. 💬 COMMANDES DE COMMUNICATION
===============================================================================

PRIVMSG <cible> :<message>
──────────────────────────
📝 Description : Envoie un message à un utilisateur ou canal
🎯 Utilisation : PRIVMSG #general :Bonjour tout le monde !
📋 Exemples :
   PRIVMSG #general :Salut !
   PRIVMSG Alice :Comment ça va ?
   PRIVMSG #aide :J'ai besoin d'aide avec IRC

NOTICE <cible> :<message>
─────────────────────────
📝 Description : Envoie une notice (message système)
🎯 Utilisation : NOTICE Alice :Message important
📋 Exemples :
   NOTICE #general :Le serveur redémarre dans 5 minutes
   NOTICE Bob :Votre compte expire bientôt
⚠️ Remarques : 
   - Les notices ne déclenchent pas de réponses automatiques
   - Utilisées pour les messages système ou administratifs

4. 🔧 COMMANDES DE MODES IRC
===============================================================================

MODE <#canal> [+/-mode] [paramètres]
────────────────────────────────────
📝 Description : Définit ou consulte les modes d'un canal
🎯 Utilisation : MODE #general +t

🎭 MODES DE CANAL DISPONIBLES :

+i (invite-only)
────────────────
📝 Description : Canal sur invitation uniquement
🎯 Utilisation : MODE #privé +i
📋 Effet : Seuls les utilisateurs invités peuvent rejoindre

+t (topic protection)
─────────────────────
📝 Description : Seuls les opérateurs peuvent changer le sujet
🎯 Utilisation : MODE #general +t
📋 Effet : Protection du topic du canal

+k <clé> (channel key)
──────────────────────
📝 Description : Définit un mot de passe pour le canal
🎯 Utilisation : MODE #privé +k motdepasse123
📋 Exemples :
   MODE #privé +k secret
   MODE #privé -k secret (pour retirer)

+o <utilisateur> (operator)
───────────────────────────
📝 Description : Donne les privilèges d'opérateur
🎯 Utilisation : MODE #general +o Alice
📋 Exemples :
   MODE #general +o Bob
   MODE #general -o Carol (pour retirer)

+l <limite> (user limit)
────────────────────────
📝 Description : Limite le nombre d'utilisateurs
🎯 Utilisation : MODE #general +l 50
📋 Exemples :
   MODE #general +l 100
   MODE #general -l (pour retirer la limite)

+m (moderated)
──────────────
📝 Description : Seuls les opérateurs/voice peuvent parler
🎯 Utilisation : MODE #general +m
📋 Effet : Canal modéré, parole restreinte

5. 🛡️ COMMANDES DE MODÉRATION
===============================================================================

KICK <#canal> <utilisateur> [raison]
────────────────────────────────────
📝 Description : Expulse un utilisateur du canal
🎯 Utilisation : KICK #general Alice :Violation des règles
📋 Exemples :
   KICK #general Bob
   KICK #general Carol :Spam
⚠️ Remarques : Nécessite les privilèges d'opérateur

MODE <#canal> +b <masque>
─────────────────────────
📝 Description : Bannit un utilisateur ou un masque
🎯 Utilisation : MODE #general +b Alice!*@*
📋 Exemples :
   MODE #general +b *!*@spam.example.com
   MODE #general +b BadUser!*@*

MODE <#canal> -b <masque>
─────────────────────────
📝 Description : Retire un bannissement
🎯 Utilisation : MODE #general -b Alice!*@*

MODE <#canal> +b
────────────────
📝 Description : Liste les bannissements actifs
🎯 Utilisation : MODE #general +b

INVITE <utilisateur> <#canal>
─────────────────────────────
📝 Description : Invite un utilisateur dans un canal
🎯 Utilisation : INVITE Alice #privé
📋 Exemples :
   INVITE Bob #vip
   INVITE Carol #équipe
⚠️ Remarques : Nécessaire pour les canaux +i (invite-only)

TOPIC <#canal> [nouveau_sujet]
──────────────────────────────
📝 Description : Définit ou consulte le sujet du canal
🎯 Utilisation : TOPIC #general :Bienvenue sur le canal général !
📋 Exemples :
   TOPIC #general
   TOPIC #général :Discussion libre - Soyez respectueux
⚠️ Remarques : Peut nécessiter le mode +t (protection topic)

6. 👑 COMMANDES D'ADMINISTRATION
===============================================================================

ADMIN <mot_de_passe>
────────────────────
📝 Description : S'authentifier comme administrateur principal
🎯 Utilisation : ADMIN admin123
📋 Exemples :
   ADMIN monmotdepasse
⚠️ Remarques : 
   - Donne les privilèges administrateur maximum
   - Mot de passe défini dans admin.conf
   - Grade 0 - Administrateur Principal 👑

OPER <mot_de_passe>
───────────────────
📝 Description : S'authentifier comme opérateur
🎯 Utilisation : OPER mypassword123
📋 Exemples :
   OPER operpass
⚠️ Remarques : 
   - Donne les privilèges d'opérateur
   - Mot de passe défini dans admin.conf
   - Grade 1 - Opérateur ⚡

GRADES
──────
📝 Description : Affiche les grades de tous les utilisateurs connectés
🎯 Utilisation : GRADES
📋 Affichage :
   👑 Admin (Grade 0) : alice
   ⚡ Opérateur (Grade 1) : bob
   🔊 Voice (Grade 2) : carol
   👤 Utilisateur (Grade 3) : dave
⚠️ Remarques : Commande disponible pour tous les utilisateurs

SETGRADE <utilisateur> <grade>
──────────────────────────────
📝 Description : Modifie le grade d'un utilisateur (admin uniquement)
🎯 Utilisation : SETGRADE alice 1
📋 Exemples :
   SETGRADE bob 0    (Promouvoir admin)
   SETGRADE carol 2  (Donner voice)
   SETGRADE dave 3   (Utilisateur normal)
⚠️ Remarques : 
   - Nécessite le grade administrateur (0)
   - Grades : 0=Admin, 1=Opérateur, 2=Voice, 3=Utilisateur

===============================================================================
7. 📊 SYSTÈME DE GRADES ET PRIVILÈGES
===============================================================================

🏆 HIÉRARCHIE DES GRADES :

Grade 0 - 👑 Administrateur Principal
─────────────────────────────────────
✅ Tous les privilèges du serveur
✅ Gérer les grades des autres utilisateurs (SETGRADE)
✅ Accès à toutes les commandes de modération
✅ Bypass des restrictions de canal
✅ Gestion des bannissements serveur

Grade 1 - ⚡ Opérateur
─────────────────────
✅ Privilèges de modération sur les canaux
✅ Commandes KICK, BAN, MODE
✅ Gestion des invitations
✅ Modification des topics protégés

Grade 2 - 🔊 Voice
──────────────────
✅ Parler dans les canaux modérés (+m)
✅ Privilèges basiques de canal
❌ Pas de commandes de modération

Grade 3 - 👤 Utilisateur
────────────────────────
✅ Utilisation basique du serveur
✅ Rejoindre les canaux publics
✅ Envoyer des messages
❌ Pas de privilèges spéciaux

8. 📁 CONFIGURATION ADMIN (admin.conf)
===============================================================================

Structure du fichier admin.conf :
─────────────────────────────────
MAIN_ADMIN=luis-fif
ADMIN_PASSWORD=admin123
OPER_PASSWORD=mypassword123
OPERATORS=bob,carol,dave
VOICE_USERS=alice,eve,frank
BANNED_USERS=spammer,troll

Paramètres :
────────────
• MAIN_ADMIN : Administrateur principal du serveur
• ADMIN_PASSWORD : Mot de passe pour la commande ADMIN
• OPER_PASSWORD : Mot de passe pour la commande OPER
• OPERATORS : Liste des opérateurs (grade 1)
• VOICE_USERS : Liste des utilisateurs voice (grade 2)
• BANNED_USERS : Liste des utilisateurs bannis

10. ⚠️ CODES D'ERREUR ET RÉPONSES
===============================================================================

Codes de réponse courants :
───────────────────────────
• 001 : Bienvenue sur le serveur
• 331 : Aucun topic défini
• 332 : Topic du canal
• 353 : Liste des utilisateurs du canal
• 366 : Fin de la liste /NAMES
• 401 : Utilisateur inexistant
• 403 : Canal interdit
• 404 : Impossible d'envoyer au canal
• 442 : Vous n'êtes pas sur ce canal
• 461 : Paramètres insuffisants
• 462 : Déjà enregistré
• 464 : Mot de passe incorrect
• 465 : Vous êtes banni du serveur
• 471 : Canal plein (+l)
• 473 : Canal sur invitation uniquement (+i)
• 474 : Banni du canal (+b)
• 475 : Clé de canal incorrecte (+k)
• 482 : Vous n'êtes pas opérateur du canal

11. 🚀 CONSEILS D'UTILISATION
===============================================================================

🔸 SÉCURITÉ :
  - Changez les mots de passe par défaut dans admin.conf
  - Utilisez des canaux +i pour les discussions privées
  - Surveillez régulièrement avec GRADES

🔸 MODÉRATION :
  - Utilisez +m pour les canaux actifs
  - Définissez des topics clairs avec +t
  - Gérez les bannissements avec parcimonie

🔸 PERFORMANCE :
  - Limitez les canaux avec +l si nécessaire
  - Utilisez NOTICE pour les messages système
  - Évitez le spam de commandes

🔸 BONNES PRATIQUES :
  - Documentez vos canaux avec des topics
  - Formez vos opérateurs aux commandes
  - Maintenez admin.conf à jour
